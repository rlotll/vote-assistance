# QA 검수 체크리스트 — 한표투표

## 1. 문서 개요

| 항목 | 내용 |
|---|---|
| 문서명 | 한표투표 검수 체크리스트 |
| 버전 | v0.1 |
| 작성일 | 2026-05-14 |
| 연관 문서 | [PRD §8](./PRD.md), [요구사항 §4](./requirements_doc.md), [ROADMAP §5.8](./ROADMAP.md) |

ROADMAP §5.8 횡단 품질 task(T-36~T-42)를 사람이 화면별로 통과/실패를 마킹할 수 있도록 분해한 체크리스트다. M3 각 화면 완료 직후 본 문서를 인쇄/공유하여 통과 여부를 기록한다.

---

## 2. 사용법

- 각 항목은 `[ ]`로 시작. 통과 시 `[x]`로 표시.
- **검증 방법** 열: `자동` = 단위/통합/E2E 테스트로 검증, `수동` = 사람 눈으로 확인.
- 화면별 적용 가능한 항목만 검수한다 (해당 없음 항목은 N/A 표기).

## 9. 화면별 추가 체크 (와이어프레임 §4 기반)

### S-01 메인
- [ ] D-day 숫자가 선거 일정 API 응답과 일치
- [ ] 비선거 기간 시 "다음 예정 선거" 안내 노출 (F-02)
- [ ] 빠른 메뉴 카드 3개 모두 탭 가능 + 라우팅 정상

### S-02 선거구 설정
- [ ] 시/도 선택 전 구/시/군 드롭다운 비활성
- [ ] 거주지 불일치 체크 시 안내 박스 즉시 노출/숨김
- [ ] 설정 완료 후 S-03으로 자동 이동

### S-03 투표소 안내
- [ ] 지도/목록 탭 전환 시 상태 보존
- [ ] 목록 항목 탭 시 해당 핀이 지도 중앙으로 이동
- [ ] 사전투표 대상자에게 사전투표소가 우선 노출되거나 명확히 구분

### S-04 후보자 비교 / S-05 정당 비교
- [ ] 분야 태그 다중 선택 가능
- [ ] 가로 스크롤 시 "좌우 스크롤" 힌트가 적절히 노출/숨김
- [ ] "원문 보기" 링크가 `target="_blank"` + `rel="noopener noreferrer"`
- [ ] 비지방선거 시 정당 탭 비활성 + 안내 메시지

### S-06 모의투표
- [ ] 라디오 단일 선택 강제
- [ ] 선택 전 "다음" 버튼 비활성
- [ ] 진행 스텝바가 현재 단계 정확히 반영
- [ ] 결과 화면에서 "다시 해보기" → 모든 선택 초기화
- [ ] 결과 화면 → "공약 비교로 이동" 시 적절한 분야로 라우팅
